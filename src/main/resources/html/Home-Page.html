About This Project

Welcome to the Story Pitch Management System! This project is designed to utilize Java as a
data and HTTP request handler. The functionality of this site itself allows users to create an
account and log-in. Beyond that each user has a role that allows the user to
create and modify stories within the rules of that role.

This website is used for the SPMS project assignment
